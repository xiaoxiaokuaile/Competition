# 2021平安产险竞赛

## 1. 思路简介
这是个典型的金融风控类题目，这是第二次参加数据竞赛，之前第一次中兴捧月竞赛中只有LGB模型没有尝试，主要是不知道LGB怎么使用，所以就参加这个比赛来学习LGB的使用方法。
